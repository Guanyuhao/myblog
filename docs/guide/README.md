# 指南

## 起步

::: warning 环境依赖
node -v
8.0 以上的版本
:::

```sh
# 安装
yarn global add vuepress # 或者：npm install -g vuepress

# 新建一个 markdown 文件
echo '# Hello VuePress!' > README.md

# 开始写作
vuepress dev .

# 构建静态文件
vuepress build .
```

在浏览器中输入http://localhost:8080/就可以看到效果了
